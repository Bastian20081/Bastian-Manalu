Tentang Game Snake

Game Snake adalah permainan klasik di mana kamu mengendalikan ular untuk memakan makanan dan tumbuh lebih panjang. Tantangannya adalah menghindari menabrak dinding atau tubuh ular sendiri.

Aturan Permainan:

  • Gunakan tombol panah atau WASD untuk mengendalikan arah ular.
  • Setiap kali ular memakan makanan, tubuhnya akan tumbuh lebih panjang.
  • Hindari menabrak dinding atau tubuh ular sendiri, jika tidak game akan berakhir!

Langkah-langkah Bermain:

  1. Tekan tombol "Mainkan Game" untuk memulai permainan.
  2. Gunakan tombol panah (atau WASD) untuk mengendalikan arah ular.
  3. Makan makanan untuk tumbuh lebih panjang dan mendapatkan poin.
  4. Hindari menabrak dinding atau tubuh ular untuk bertahan hidup.
  5. Cobalah untuk mendapatkan skor setinggi mungkin!
